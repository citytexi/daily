# 클라이언트-서버 패턴 - Client-server pattern

## 기본 설명

---

- 하나의 서버 컴포넌트와 다수의 클라이언트 컴포넌트로 구성
- 클라이언트가 서버에 서비스를 요청하면 서버는 클라이언트에게 적절한 서비스 제공
- 서버는 클라이언트로부터의 요청을 계속 대기한다

### 컴포넌트 (Component)

---

- 독립적인 업무 또는 기능을 수행하는 실행 코드 기반 모듈
